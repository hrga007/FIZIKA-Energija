# Fizika radionica

Interaktivna web radionica za osnovnoškolsku fiziku: rad, snaga, vrste energije, unutarnja energija, toplina i toplinsko rastezanje tekućina.

## Pokretanje

```bash
npm install
npm run dev
```

## Što je uključeno

- mentalna mapa pojmova
- formula kartice s primjerima, jedinicama i čestim greškama
- interaktivni laboratoriji s klizačima
- rješavač zadataka korak po korak
- kviz u tri razine s objašnjenjima
- brzi šalabahter i pretvorbe jedinica
- lokalno spremanje napretka u `localStorage`
